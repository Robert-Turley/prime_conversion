Prime conversion relies on a unique method for interpreting Real Numbers by utilizing Pi (3.141...) as the base value of the language.


Rather than distinguishing the values Pi and 1 as Irrational/Rational numbers, Prime conversion equates these two values using three arguments:



I == 00

II == 0

III == 000


Where I equals Pi and 0 equals Tau (2Pi). 00 (represented as [00) in some cases) also refers to Absolute Zero (0K = -273.15Â°C). Prime notation does not rely on the traditional four-quadrant (Cartesian) system.



These three arguments establish the algorithms below, allowing the user to divide any whole number by two and still get a whole number!


If EVEN, (N/2) until reaching 3, 2 or 1

If ODD, (N+3)/2 until reaching 3, 2 or 1


The final value is I or 0 depending on its step in the binary* conversion process.



EXAMPLE


Binary Conversion Whole Number Tally Notation
I 25 (Starting value) IIIIIIIIIIIIIIIIIIIIIIIII
0 14 ((25+3)/2) 00000000000000
I 7 (14/2) IIIIIII
0 5 ((7+3)/2) 00000
I 4 ((5+3)/2) IIII
0 2 (4/2) 00
I 1 (2/2) I

*Binary Conversion does not directly reference the traditional computer language of binary, rather the conversion process from Pi(I) to Tau(0) and vice versa based on the three arguments.


The same process can be applied to number sequences, like the famous Fibonacci sequence!

Doing so provides a (theorized) infinitely recurring 8-loop:
I I 0 III 0 0 I III
